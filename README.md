# Mon App Habitudes

Application mobile développée avec [React Native (Expo)] permettant de suivre ses habitudes quotidiennes.

## Fonctionnalités

- Ajouter une habitude (nom, catégorie, objectif, note (facultatif))
- Liste des habitudes
- Marquer une habitude comme faite
- Suivi du nombre de répétitions
- Modifier une habitude
- Supprimer une habitude
- Sauvegarde locale (AsyncStorage)
- Statistiques (à venir / en développement)

## Lancer le projet en local

### 1. Installer les dépendances

npm install

### 2.Démarrer le projet Expo

npx expo start
